# system/

Framework documentation for the Second Brain vault — templates, examples, and setup guide.

This folder is for humans, not agents. Agent definitions and commands live in `.claude/`.

---

## Structure

```
system/
├── templates/    — Note templates (state files, daily notes, claude-ai project)
├── examples/     — Filled-in examples for reference
├── README.md     — This file
└── SETUP.md      — Setup guide for new users
```

---

## Templates

| Template | Use when |
|---|---|
| `project-state-template.md` | Creating a new project state file |
| `area-state-template.md` | Creating a new area state file |
| `daily-note-template.md` | Creating a daily journal entry |
| `claude-ai-project-template.md` | Setting up a Claude.ai mobile workspace |

Agent definitions, skill files, and slash commands → `.claude/`
