# Archive

Completed projects and retired areas — moved here when work is done or no longer active.

---

## Structure

```
Archive/
└── Project Name/
    ├── state-project-name.md   ← final state file (moved from Projects/)
    └── summary.md              ← closing summary — what was built, learned, decided
```

Mirror the original folder name from `Projects/` or `Areas/` exactly.

---

## summary.md — what to include

| Section | Content |
|---|---|
| **Outcome** | What was actually delivered vs. the original Definition of Done |
| **Key decisions** | Major choices made and why |
| **Lessons learned** | What to carry forward to future projects |
| **Related wiki pages** | Links to `wiki/` entries created during this project |
| **Related journal** | Date range of relevant daily entries |

---

## When to archive

- Project reaches Definition of Done → move `Projects/[name]/` here, add `summary.md`
- Area becomes inactive (no action for 6+ months, no longer relevant) → move `Areas/[name]/` here
- Never delete — Archive is permanent record, not trash

---

## Privacy

All content in this folder is gitignored — nothing in `Archive/` leaves your machine.
