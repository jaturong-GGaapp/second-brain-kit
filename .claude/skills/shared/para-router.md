# PARA Router

**Role:** classify content → route ไปยัง Projects / Areas / Resources / Archive

## PARA Definition (สำหรับ Gap's vault)

| Category | นิยาม | ตัวอย่าง |
|---|---|---|
| Projects | มี end date + outcome ชัดเจน | App Startup, พัฒนาที่ดินไทย |
| Areas | ongoing, ไม่มี end date | การเงิน, สุขภาพ, พัฒนาตนเอง |
| Resources | reference material สำหรับอนาคต | แผนเกษียณ, learning notes |
| Archive | เสร็จแล้ว หรือไม่ active | — |
